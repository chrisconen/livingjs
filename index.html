Architecture

Three pillars. Zero complexity.

Observer watches. Brain decides. Actor executes. The whole loop runs in the browser — no server, no latency, no cost.

👁
Observer

Passive behavioral tracking. Knows everything about the visitor without asking a single question.

  • Scroll depth & velocity
  • Click patterns & targets
  • Time on sections
  • Device, hour, referral source
  • Return visit detection
  • Engagement score 0–100
🧠
Brain

Rule engine handles 80% of decisions instantly. AI handles the rest. Everything gets cached.

  • 8 built-in behavioral rules
  • Intent classification
  • Urgency scoring 1–10
  • AI fallback (Claude / GPT)
  • Signal fingerprint caching
  • Industry-aware context
⚡
Actor

Makes it happen. Morphs the page, shows modals, calls external services via MCP.

  • Swap CTAs dynamically
  • Show/hide relevant content
  • Emergency & exit modals
  • MCP tool execution
  • Webhook integration
  • Smooth DOM transitions

Behaviors

Pre-built intelligence. One word to activate.

Each behavior is a composable module. Enable what you need. They don't conflict.

Emergency Detect

Phone clicks + after hours + deep scroll = surface emergency callback form with SMS notification to owner.

urgency: 10

Lead Qualify

Smart multi-step qualification based on behavioral signals. No chatbot. No forms. The page itself asks the right questions.

conversion

After Hours

Different UX outside business hours. Surfaces online booking, hides phone CTAs, adapts messaging to "closed but available."

24/7

Exit Intent

Detects engaged visitors about to leave. Offers the right incentive based on their browsing pattern — not a generic popup.

retention

Return Visitor

Remembers returning visitors. Changes CTAs from "Learn More" to "Welcome Back — Ready to Get Started?" on visit #3+.

personalization

MCP Native

Book directly to Google Calendar, charge via Stripe, send confirmations via Gmail — all through Model Context Protocol.

integration
